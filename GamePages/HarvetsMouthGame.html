Harvest Mouth

About Project

This was made during a 48 hour game jam with 4 other people, for this jam the theme was harvest. You're a hamster who works at a mortuary and steals organs and puts them in your cheeks. These organs aren't easy to get out so different tools need to be used, just make sure you don’t get caught in the act since the supervisor has a close eye on you.

Mini-games

Similar to how easy it was to add effects to BugBug’s effect system, I wanted to be able to add mini-games easily. That's why I decided to support adding mini-games throughout development by making a base class for all mini-games, they only needed to be inherited by the base class and they would automatically work. Only the mini-game had to be implemented since both the base class and other systems handled the rest like what would happen after losing or winning the mini-game. In total I created 5 out of 6 mini-games which took up most of the time for this game jam.
