You Oar Me

About Project

During this game jam my group of 3 people decided to only use software we had no experience with, for me that meant using Godot for the first time. We really wanted to challenge ourselves but also learn more during this jam so that’s why we made the decision. It was really hard in the beginning but after just a couple of hours we got the hang of it and created this game. This time the theme was “You or me” so we decided to make a racing game where you steer a boat and try to get to the finish line before the ghost boat does. The twist is that the lap gets saved every time someone reaches the goal, this lap is then used for the ghost boat the next time someone plays the game.

Ghost Boat

The ghost boat is the player's enemy, it uses the fastest lap out of any player and if a player is faster than the ghost boat then that lap gets saved. This is done by saving the player position every second or so to a list. When a player reaches the goal and their time was faster than the ghost boat then the list of positions along with the lap time gets serialized to a file using Json. The ghost boat can then load the positions and lerp between the points to recreate the path that the player took.
